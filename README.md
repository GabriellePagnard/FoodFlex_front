# 🥗 Foodflex 🥗

## 📜 Présentation

Bienvenue sur Foodflex, votre assistant culinaire personnel pour une vie plus organisée et savoureuse ! 🍽️   

<br>

Foodflex est une application qui vous permet de planifier vos repas hebdomadaires en toute simplicité. Explorez un vaste catalogue de recettes, créez votre planning de repas personnalisé, et sauvegardez vos plats préférés en favoris pour un accès rapide. Avec Foodflex, gagnez du temps et de l'inspiration pour bien manger chaque jour ! 🕒👨‍🍳   

<br>
<br>

## ✨ Fonctionnalités Principales

🗂️ Catalogue de recettes : Accédez à plus de 450 recettes variées et inspirantes   
🔍 Filtrage et recherche avancés : Trouvez des recettes en fonction du nom, des ingrédients ou des catégories   
💾 Gestion des favoris : Ajoutez vos recettes préférées pour un accès simplifié   
📅 Planification des repas : Organisez vos repas de la semaine en attribuant des recettes à chaque jour   
🎲 Génération aléatoire : Génération automatique de menus hebdomadaires pour les plus pressés   
👤 Profil utilisateur : Gérez vos informations personnelles et accédez à votre espace dédié   
🛒 Liste de courses : Génération automatique d'une liste de courses en fonction de votre planning de repas

## 🔧 Technologies Utilisées

⚛️ React : Construction de l'interface utilisateur   
🎨 Tailwind CSS : Framework CSS pour un design moderne et responsive   
🌐 Redux avec Redux Toolkit : Gestion centralisée des états pour une interaction fluide entre les fonctionnalités   
💻 Vite : Outil de build rapide pour le front-end   
🔤 TypeScript : Typage statique pour un code robuste et maintenable   
🍲 API RESTful : Interaction avec le backend pour la gestion des recettes et des utilisateurs   

<br>
<br>

## 📂 Structure du Projet

src/components/ : Contient les composants principaux tels que la planification des repas, la gestion des favoris, et le catalogue de recettes   
src/hooks/ : Regroupe les hooks personnalisés pour gérer les données et interactions   
src/store/ : Configuration de Redux et gestion des états globaux   
src/styles/ : Contient les styles globaux et personnalisés

## 🛠️ Installation et Utilisation

Pour installer et exécuter Foodflex localement, suivez ces étapes :   

➡️ Clonez le dépôt :

```bash
git clone https://github.com/yourusername/Foodflex.git
```

➡️ Accédez au dossier du projet :

```bash
cd Foodflex
```

➡️ Installez les dépendances :

```bash
npm install
```

➡️ Démarrez le serveur de développement :

```bash
npm run dev
```

➡️ Ouvrez votre navigateur et accédez à : http://localhost:5173
